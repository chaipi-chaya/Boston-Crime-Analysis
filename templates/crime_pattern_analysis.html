Pattern analysis using Markov Chain¶

Markov chain is a model used for describing a sequence of possible events that depends on a previous event. We will give an example with our analysis.
We can use the Markov chain to create to nodes of districts and probabilities that the state will move to the next node (crime happen on the next district)

For more insightful analysis, we will create a Markov chain for groups of crime, instead of overall crime.

def Markov_chain_analysis(t, des):
    if t == 'description':
        t = 'OFFENSE_DESCRIPTION'
    else:
        t = 'OFFENSE_CODE_GROUP'
    
    # get dataframe
    crime = crimeandweather[crimeandweather[t] == des]
    crime.index = pd.to_datetime(crime['OCCURRED_ON_DATE'])
    crime = crime.sort_index(axis = 0) 
    # create list of dic for count
    districtdict = {}
    uniquedistrictlist = crime['DISTRICT'].unique().tolist()
    for i in uniquedistrictlist:
        districtdict[i] = []
        
    # add next district to current district
    alldistrictlist = crime['DISTRICT'].tolist()
    for i in range(0,len(alldistrictlist)-1):
        districtdict[alldistrictlist[i]].append(alldistrictlist[i+1])
        
    # count duplicate in list and sum for cal prob
    countdistrictdict = {}
    for i in districtdict.keys():
        countdistrictdict[i] = {}
        countdistrictdict[i]['sum'] = 0
        for j in districtdict[i]:
            if j in countdistrictdict[i].keys():
                countdistrictdict[i][j] += 1
            else :
                countdistrictdict[i][j] = 1
            countdistrictdict[i]['sum'] += 1
            
    # change count them to prob
    probdistrictdict = {}
    for i in countdistrictdict.keys():
        probdistrictdict[i] = {}
        for j in countdistrictdict[i].keys():
            if countdistrictdict[i]['sum'] == 0:
                probdistrictdict[i][j] = 0
            else:
                probdistrictdict[i][j] =  round(countdistrictdict[i][j]/countdistrictdict[i]['sum'],4)
        probdistrictdict[i].pop('sum')
    
    return probdistrictdict

What this function does is it goes through specific crime records and counts the next districts. For example, if the first bank robbery happens in A1, and the next one is in B2. For A1 node, the value of B2 is 1. If the next crime happen in C11, A1 node has 1 B2, and B2 node has 1 C11. The process repeats until it reaches the end. Then it will calculate the probability of the next node. For instead, start with A1, there is 5 B2 and 5 C11, so the probability that the bank robbery happened in B2 after A1 is 0.5

bankrobcrime = Markov_chain_analysis('description', 'ROBBERY - BANK')

import matplotlib.pyplot as plt
%matplotlib inline

# plot prob
cols = 3
rows = 4
fig, ax = plt.subplots(4, 3, figsize=(20,20))
key = list(sorted(bankrobcrime))
n = 0
for i in range(0, rows):
    for j in range(0, cols):
        if i == 3 and j == 2:
            break
        index = j + i-1
        lists = sorted(bankrobcrime[key[n]].items(), key = lambda x : x[1], reverse=True)
        x, y = zip(*lists)
        ax[i, j].bar(x, y)
        ax[i, j].set_title(key[n])
        n += 1

We can see from the bar graph that in if the bank robbery happens in E18 district, the probability of the next bank robbery happens in C6 district is 0.3

However, this model is not always valid.
There is a limitation of the model which depend on sample size and the probability of the event. We will not go into detail how to check the validity of the model. But, when we use the model with the large sample size, the model is mostly valid.
